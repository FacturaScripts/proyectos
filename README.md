# Proyectos
El plugin Proyectos te permite gestionar proyectos y sus fases en un único lugar.
Crea proyectos con tareas, responsables y presupuesto.
Registra tiempos y gastos por tarea, asocia pedidos y facturas al proyecto y controla
el avance mediante estados y porcentajes de completado.
Centraliza la planificación, el control y la facturación para mejorar el seguimiento y la toma de decisiones.

Plugin para gestionar proyectos en FacturaScripts.
- https://facturascripts.com/plugins/proyectos
